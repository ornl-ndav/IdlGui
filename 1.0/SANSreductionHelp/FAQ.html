  • LOAD
    • The run number I'm looking for can not be found: that problem can be caused by various factor. #1, make sure the program findnexus is in your path. For that, check the top of the LogBook and see if the program found it. If it says FAILED next to findnexus, then you know what to do, install findnexus (contact pf9@ornl.gov if you want more information about findnexus). #2, your file is not where is should be. findenxus looks for the NeXus files only in a few specic places. If you want findnexus to systematically look your NeXus files in a specific folder, just let me know and I will add this option (j35@ornl.gov).
  • LOG BOOK
    • What is exactly the SEND TO GEEK button? When using this button, a copy of the log book plus a few information about the current session is sent to j35@ornl.gov. This allows us to get a copy of your session and see what went wrong and help us in solving the issue you had. Do not hesitate to add a message to help us solving the problem. Of course, you can also used this button to tell us how much you love the application ;-).

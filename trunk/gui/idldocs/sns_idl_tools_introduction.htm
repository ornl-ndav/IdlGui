The sns_idl_tools package is available from the following analysis computer:
  • heater
  • bac
  • lrac
  • mrac

To reach the sns_idl_tools portal, just type sns_idl_tools 


          >sns_idl_tools

An IDL virtual machine flash screen appears.


Just click anywhere in this window to launch the sns_idl_tools portal.

Select an IDL tool by clicking the list box below the label Select your IDL tool

The Description box gives a brief overview of the tool selected and the preview box (on the right) gives a preview of that same tool. According to the analysize computer from where you sstarted the application, you have access or not to the tool selected. If you don't have access, the launch application button is not activated.

How to Create a NeXus File Step by Step

  1. Load one or a set of runs

    You can either load 1 run at a time or enter a set of runs using the following nomenclature:

    • 1454 -> run 1454
    • 1454,1458 -> run 1454 and run 1458
    • 1454-1457 -> run 1454, run 1455, run 1456 and run 1457
    • 1454,1456-1458 -> run 1454, run 1456, run 1457 and run 1458

    You can speed up the search process by selecting the proposal folder from where this run belongs.
    Examples of Inputs

    Press ENTER or select an instrument to allow the program to localize the run(s). The progam looks for the runs of interest and display the result of the search in the status window.
           
    Search process in progress and result

  2. Check and change, if necessary, the histogram parameters The time offset, the maximum time, the bin widths and the type of bining parameters of the first file found are display and will be used for all the runs selected. These default parameters have been defined on the DAS.


  3. Set the output folders Modify the default output folder if necessary and and select Copy -> Instrument Shared Folder and/or Copy -> Proposal Shared Folder if necessary.


  4. Run the translation Press the CREATE NEXUS button to run the translation. The translation is divided into 4 steps. The translation is successful only if all 4 steps ran with success.
    A progress bar informs the user of the status of the translation.
           
    Once the translation is done, the status message gives the list of NeXus files created and their final location.

Menu: Live Infos

Gives live information about the cursor position or current selection.

  • Show Cursor Informations
  • Show Counts vs xaxis at cursor y position. Plot displays the integrated counts over the y axis at the given position of the cursor.
  • Show Counts vs yaxis at cursor x position. Plot displays the integrated counts over the x axis at the given position of the cursor.

Y axis of the two plots (counts vs Qx and counts vs Qz) can be either linear or logarithmic.

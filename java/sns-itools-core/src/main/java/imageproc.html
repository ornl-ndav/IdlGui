Advanced Examples: ION Java Low-Level Classes
This example shows an image processing applet built with the ION Java Low-Level Classes.
Java virtual machine failed to start. Is Java enabled in your browser?
Applet not visible? View the ION Java troubleshooting page.
Controls:
  • Select Operation-- Select the radio button to specify how to process the image.
  • Select a Region of Interest (ROI)-- Click on the main image area to select the center of a square ROI.

    • This applet integrates Java and IDL to provide an interface for processing square Regions of Interest on an image. When the user clicks on the image, a square area centered on the clicked location appears in the zoom window and is processed by one of the available algorithms.
